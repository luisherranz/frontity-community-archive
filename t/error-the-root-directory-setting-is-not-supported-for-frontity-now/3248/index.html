luisherranz 30 October 2020 11:55 4

If your Frontity project is in a different folder, you can do this:

github.com

luisherranz/next-and-frontity/blob/master/now.json

{
  "version": 2,
  "builds": [
    {
      "src": "package.json",
      "use": "@now/next"
    },
    {
      "src": "blog/package.json",
      "use": "@frontity/now"
    }
  ]
}

Also, make sure to configure your subdirectory setting both in Frontity and WordPress:

github.com

luisherranz/next-and-frontity/blob/master/blog/frontity.settings.js#L30-L37

    

  1.   name: "@frontity/wp-source",
    2. 

  2.   state: {
    3. 

  3.     source: {
    4. 

  4.       api: "https://test.frontity.io/wp-json",
    5. 

  5.       subdirectory: "blog"
    6. 

  6.     }
    7. 

  7.   }
    8. 

  8. },
    9.

Let us know if that solves your question.
