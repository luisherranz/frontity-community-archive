Whilst not a fix per-se, I did manage to suppress all of these errors by updating my .eslintrc file:

{
  "extends": [
    "wesbos"
  ],
  "rules": {
    "import/no-extraneous-dependencies": 0,
    "import/no-unresolved": 0
  }
}
