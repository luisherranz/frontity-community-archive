1.0 Roadmap

  • html2react

    • Implement the package.
    • Include it in Mars Theme.
    • Create the Image component

  • mars-theme

    • Add AMP support.
    • Fix accessibility issues.
    • Fix WP CSS core issues.
    • Add featured image setting.
    • Add authors pages.
    • Add loading components.

  • wp-source

    • Move list pages to their own data entry.
    • Change .data() api to .get().
    • Remove WP domain from REST API link fields.
    • Move routeToParams logic to source packages.
    • Add setting to allow WP installed in subfolders.
    • Add setting to allow WP that changed their category or tag urls.
    • Add setting to add custom post types.
    • Add setting to add custom taxonomies.

  • core

    • Simplify modes and entry-points.
    • Merge state arrays instead of replace.

  • connect

    • Add HMR.

  • tiny-router

    • Add the auto-fetch feature.

  • Docs

    • - Add an “Understanding Mars Theme” guide.
    • - Add a “Learning Frontity” section with the framework concepts.

Let’s go!
