orballo 18 May 2022 11:13 2

Hi @nicobava96 !

To install dependencies on your package, you need to add them manually to the package.json file and then run npm i from the project root folder.

You should not run npm i inside a package, because it can create conflicts between dependencies, and the way Frontity works, all packages dependencies are installed together in the node_modules of the project’s root folder.
