New Docs: Frontity CLI + SEO + Showcase sections

📢 Announcements
Reyes 27 May 2020 18:27 1

We’re happy to announce that we have recently published new Frontity documentation, featuring:

  • CLI Commands: how to use Frontity CLI and a list of the commands you have available.
  • SEO: how to customize meta tags and robots.txt in Frontity.
  • Showcases: a new section to share the awesome projects the community is building with Frontity. This will help us and others learn, get inspired and see what can be accomplished with the framework. We’d love to see your Frontity site in the list!

Update: the Showcase page has been moved to https://frontity.org/showcase.

We’ll keep improving and iterating on the documentation. Feel free to let us know if you have any feedback about the new pages.

Community contributions to the documentation are always welcome. You can read more on how to contribute here.
