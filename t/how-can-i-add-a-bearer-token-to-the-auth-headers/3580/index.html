The code you share seems to be right. For a token for example like sdf98sd7f9sf7s98f7sd9f87sd98fsd the code will be

{
  name: "@frontity/wp-source",
  state: {
    source: { 
      api: .....
      params: {
        per_page: 30,
        type: ["post", "page"],            
      },
     auth: 'Bearer sdf98sd7f9sf7s98f7sd9f87sd98fsd'
