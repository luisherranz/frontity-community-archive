phn 27 February 2021 19:24 6

Hi @gallonjuanito,

I use the following setting in my frontity.settings.js to set /blog as my frontpage:

{
      "name": "@frontity/wp-source",
      "state": {
        "source": {
          "api": "https://wp.mydomain/wp-json",
          "homepage": "/home",
          "postsPage": "/blog",
        }
      }
    },

I hope this helps
