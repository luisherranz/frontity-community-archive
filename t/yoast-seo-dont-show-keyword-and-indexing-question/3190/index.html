Hi @bastien

Welcome to the Community!

bastien:

1- why frontity with Yoast Seo and head tag plugin does not show my keyword

In order to work properly, @frontity/head-tags package requires also having the REST API – Head Tags plugin installed in your WordPress installation. Having this is enough for Frontity to display all the SEO metadata (YOAST ones included) in the head of your HTML

If this is not working for you, can you please provide a repo or code-sandbox with your code? This is especially helpful to find solutions of technical issues with specific code

Detailing the info suggested here when having issues will help the community to provide the best possible help as quickly and as efficiently as possible.

bastien:

2- how will you manage? the google indexing system.
are you putting the worpress site in private?

Can you elaborate this question a but more?

Frontity uses Dynamic SSR which means it will generate HTML from your React App and the data in your REST API for each page. This will make your pages SEO friendly as the content of your pages will be optimized for crawlers of search engines.

Besides this, Frontity packages like @frontity/head-tags improve even more the SEO of your pages by adding SEO metadata to the head of your HTML

Here you have more info about SEO in Frontity projects → https://docs.frontity.org/guides/seo
