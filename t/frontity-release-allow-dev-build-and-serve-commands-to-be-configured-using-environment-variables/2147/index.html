Frontity release: Allow dev, build and serve commands to be configured using environment variables

👨‍💻 Framework Development Releases
SantosGuillamot 19 June 2020 10:35 1

New features

  • Allow dev , build and serve commands to be configured using environment variables (feature discussion).

Bug fixes

  • Fix a bug that was causing pageviews not to be sent if the previous page has the same title #471. This bug affects the following packages:
    • @frontity/analytics

Enhancements

  • Do not allow the new state.frontity.debug flag to be true in production. #467
  • We have added proper readmes to the Frontity packages:
    • frontity
    • @frontity/wp-source
    • @frontity/analytics
    • babel-plugin-frontity
    • @frontity/components
    • @frontity/comscore-analytics
    • @frontity/connect
    • @frontity/core
    • create-frontity
    • @frontity/error
    • @frontity/file-settings
    • @frontity/head-tags
    • @frontity/hooks
    • @frontity/html2react
    • @frontity/mars-theme
    • @frontity/router
    • @frontity/source
    • @frontity/tiny-router
    • @frontity/twentytwenty-theme
    • @frontity/types
