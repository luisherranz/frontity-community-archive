It will work as long as you put the Frontity project inside the /blog folder and add this now.json file to the root: { "version": 2, "builds": [ { "src": "package.json", "use": "@now/next" }, { "src": "blog/package.json", "use": "@frontity/now" } ] } I’ve created a the repo example: https://next-and-frontity.luisherranz.now.sh/ And this is the deploy: https://next-and-frontity.luisherranz.now.sh/

Here’s the repo with a Next App including a Frontity Blog → https://github.com/luisherranz/next-and-frontity
And here’s the deploy of this project→ https://next-and-frontity.luisherranz.now.sh/

Hope this helps
