Frontity release: Make @frontity/mars-theme AMP-aware

👨‍💻 Framework Development Releases
SantosGuillamot 2 July 2021 11:21 1

Features

  • As part of the AMP feature, we have made @frontity/mars-theme package AMP-aware. This means that, if you are using the @frontity/amp package and you install this theme, it just works. #825

Fixes

  • Fix problem with CDNs that return the same HTML response for URLs with same queries in different order. #850
  • Add the .DS_Store files to the list of files that a directory where you want to run npx frontity create can contain. #841

Credits

Huge thanks to @michalczaplinski and @luisherranz for helping on this release!
