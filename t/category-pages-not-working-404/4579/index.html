mburridge 24 May 2021 15:38 12

Hi @rob_cal_uk

I’ve cloned your repo and managed to get it working.

Try adding a categoryBase property to the wp-source settings in frontity.settings.js.

{
      "name": "@frontity/wp-source",
      "state": {
        "source": {
          "url": "https://wp.robcallaghan.co.uk",
          "categoryBase": "/blog/category"
        }
      }
    }

The value here will be what you define in Settings → Permalinks → Category base in the WordPress admin.

Let me know how you get on. For more info see here.

mburridge 25 May 2021 08:18 14

Hi @rob_cal_uk

There isn’t a default for categoryBase. If you leave it out of your configuration in frontity.settings.js and look at the state you’ll see that it’s just an empty string. I think the problem was that you had the Category base set in the WP admin, so you also needed it in your Frontity configuration.

I’m glad you managed to get it working.
