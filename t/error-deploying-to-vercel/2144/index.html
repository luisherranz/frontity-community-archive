To “clean” your node_modules and your package-lock.json from dependencies that are not being used in your project you can do…

rm -rf node_modules && rm -rf package-lock.json && rm -rf yarn-error.log && rm -rf yarn.lock && npm cache clean --force

and then do npm install or yarn install depending on which package manager you want to use
