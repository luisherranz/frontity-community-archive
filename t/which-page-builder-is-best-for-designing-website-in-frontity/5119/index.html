Johan 11 August 2021 17:57 2

Hi, Welcome to the Frontity forums.

To answer your question; if you’re looking for a page builder like Oxygen, Elementor or Divi, then you’re out of luck.
They can be made to work with Frontity, but will require a lot of coding to make it work, or worse, build everything twice.

Frontity is designed to work on it’s own templates which are built in React and only retrieve raw data from WordPress.
Content from Gutenberg does work, and as said before can be made to work with other page builders, but it won’t be as easy as plug-and-play.

Frontity is a React-based framework for building fast and scalable WordPress websites. While Frontity itself does not include a page builder, you can use any React-based page builder with Frontity to design your website.

Here are a few popular React-based page builders that you can consider for use with Frontity:

  1. Gatsby: Gatsby is a popular React-based static site generator that allows you to create fast and optimized websites. It offers a variety of plugins and themes, including several page builder plugins, such as Gatsby Plugin Page Creator and Gatsby Plugin Layout.
  2. Next.js: Next.js is a React-based framework that allows you to build server-rendered web applications. It includes several features for building static and dynamic websites, and you can use any React-based page builder with Next.js to design your website.
  3. React Page Scroller: React Page Scroller is a lightweight and customizable page builder that allows you to create smooth-scrolling websites with dynamic content. It offers a variety of components, such as full-screen sliders and parallax effects, that you can use to create a visually appealing website.

Ultimately, the best page builder for your Frontity website will depend on your specific needs and preferences. It’s a good idea to research and try out different options to determine which one works best for you.

Hello Michel,

For designing a Frontity website, you may not necessarily need a traditional page builder as Frontity utilizes React, which works differently than typical WordPress page builders. Frontity integrates with WordPress via the WordPress REST API, allowing you to create themes with React. I’d recommend leveraging React components and libraries for designing and structuring your pages.

When it comes to SEO plugins for WordPress, Yoast SEO and All in One SEO Pack are widely recognized and well-maintained plugins. They are user-friendly and offer comprehensive solutions for your on-site SEO needs, including meta tags, sitemaps, and content analysis.

If you’re looking for React libraries that could help with SEO, consider React Helmet. It allows you to manage the meta tags of your pages, which is crucial for SEO.
