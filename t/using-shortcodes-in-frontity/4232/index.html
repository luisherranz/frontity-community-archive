Hi @mh123 ,

Shortcodes in WordPress are just quick ways of generating HTML The result of the shortcodes is included in the rendered content available for posts via the REST API

Regarding Contact Form there are some updates pending to be done in the package (not maintained by the Frontity team) so that may lead to some issues. Have a look at the following post to learn more about this

Contact Form 7 Throws Error on Submit 🙋Get Help
Hi again @mh123 , After another review, I’ve realized that the contact form 7 package from NPM doesn’t work with the latest version of the contact form 7 plugin. So, you should either get the package from Github, or use an earlier version of the plugin. Check this post for more info about this

Hope this helps
