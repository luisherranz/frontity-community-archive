Hi again!

I think you misunderstood how Frontity works. It needs to run in another server different than your WordPress and Node must be installed in that server. Frontity doesn’t interact with your WP for anything else than requesting data from the REST API.

smvmobi:

Mas o Frontity pelo que entendi, após a instalação do mesmo eu posso trabalhar através do cms do próprio wordpress, correto!

…sem ter que estar criando pelo terminal com o react ou JS

I’m afraid that’s not correct. You DO need to deal with the terminal and code. You can’t make changes in Frontity’s themes from WP Admin.
