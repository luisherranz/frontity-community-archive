Hi,

I released a slight change to the Frontity Contact Form 7 package now. I noticed that if you had the plugin Akismet active on your wp site, a script is added to forms that would fail, and the page with the form would fail to render. So I made a small change to ensure that the Akismet plugin can be active and do its job.

npm

@aamodtgroup/frontity-contact-form-7

Contact Form 7 extension for Frontity theme.

EDIT:

I have also added some new functionality. If you want to reset the form when a user has started to fill out a form, but does not finish it, and leave the page, you now can.
