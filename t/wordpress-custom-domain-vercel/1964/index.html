Johan 26 July 2021 17:12 14

I checked the DNS settings of those domains and I see the following:

  • cristhianjhl.com has an A-record to an IP address of Vercel (correct)
  • www.cristhianjhl.com has a CNAME-record to the same server (correct)
  • wp.cristhianjhl.com has a CNAME-record to the same server (incorrect)

So you have to change the DNS settings for wp.cristhianjhl.com so it has an A-record to the IP address of your PHP server.
