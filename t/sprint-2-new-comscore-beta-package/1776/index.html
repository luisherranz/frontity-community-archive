Sprint 2 Review

This is the summary of the Sprint 2. Bear in mind that, as it was the first one estimating story points and we don’t know yet how many we’re able to do in one sprint, we decided to include a lot of features to start learning. This is a summary:

  • Planned - 74pt
    • Done: 11pt.
    • In progress: 37pt. We estimate that 28pt aprox are already done but not released.
    • Not started: 35pt.

Released

You can find the information of the last release at its announcement, and we had to do this another release at the beginning of the sprint to fix a critical bug.
