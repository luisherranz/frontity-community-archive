Sprint 7 planning

This sprint will go from the 2nd September to the 30th September. This is going to be our focus, in this order:

  • Finish all the features that are currently in progress: New action to post comments and @frontity/yoast package.
  • Develop a functional version of the preview.
  • Support for custom headers in Source packages, which is directly related to the preview.
  • Develop the SmartAds package.
  • TypeScript 3.9 breaks @frontity/source types.

Apart from this, we’ll do the research of the Server Extensibility during this sprint.

Included in the sprint :

  • All the issues that were in progress in the Sprint 6.
  • Research of the Server Extensibility.
  • Finish what is left to have a functional preview:
    • Finish the PHP plugin for the embedded mode and the preview.
    • Add e2e for the preview.
  • Support for custom headers in Source packages, which is directly related to the preview.
  • Smart AdServer package.
  • TypeScript 3.9 breaks @frontity/source types [8pt].

Backlog:

  • AMP beta version.
  • Research of the Frontity Hooks.

You can keep track of the process at the Sprint board.
